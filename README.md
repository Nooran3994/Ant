
  # ANT - Adaptive Nesting Technologies

> **"To provide world-class solutions in agriculture and beyond through innovative hardware, software, and services."**

ANT (Adaptive Nesting Technologies) is a mission-driven technology company dedicated to empowering farmers through cutting-edge, sustainable solutions. We bridge the gap between traditional farming and modern technology to increase food security and farmer profitability.

---

## 🌟 Vision & Mission

### Purpose
Empower farmers by providing affordable, efficient, and sustainable farm solutions that reduce farming expenses, increase food security, and improve profitability.

### Mission
To provide world-class solutions in agriculture and beyond through innovative hardware, software, and services.

### Vision
Creating a global ecosystem where innovative technologies empower farmers, minimize expenses, and improve farm efficiency.

## 🚀 Core Product: JotoPro

**JotoPro** is our flagship solar-powered smart brooder designed to revolutionize poultry farming.

-   **Precision Control**: Automated temperature and environment management for optimal poultry growth.
-   **Sustainability**: Powered by solar energy, reducing operational costs and carbon footprint.
-   **Impact Driven**: Designed for efficiency in diverse agricultural environments.
-   **Certification**: **Certified by GreenLabs** for impact-driven innovation.

## 🛠️ Service Ecosystem

Beyond hardware, ANT provides a comprehensive digital suite for modern agriculture:

1.  **Poultry Advisory Tools**: Expert guidance and data-driven insights for flock management.
2.  **Credit Scoring**: Financial inclusion tools to help farmers access capital based on their farming performance.
3.  **Market Linkages**: Connecting farmers directly to buyers, ensuring fair pricing and reduced waste.

## 💻 Technology Stack

This project is built using modern, high-performance web technologies:

-   **Framework**: [React 18](https://reactjs.org/)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
-   **UI Components**: [Radix UI](https://www.radix-ui.com/) (Primitives for accessibility and styling)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Routing**: [React Router DOM](https://reactrouter.com/)
-   **SEO**: [React Helmet Async](https://github.com/staylor/react-helmet-async) for meta tag optimization.
-   **Data Vis**: [Recharts](https://recharts.org/) for analytics and reporting.

---

## 🏗️ Project Architecture

The application follows a modular, component-based architecture:

-   **`/src/components`**: Reusable UI components and page-specific sections (Hero, Solutions, Team, etc.).
-   **`/src/components/ui`**: Base primitive components built on Radix UI.
-   **`/src/App.tsx`**: Main entry point and routing logic.
-   **SEO Optimized**: Includes pre-configured `sitemap.xml` and `robots.txt` for maximum discoverability.

## 🚦 Getting Started

### Prerequisites
-   [Node.js](https://nodejs.org/) (Version 18 or higher recommended)
-   [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
1.  Clone the repository:
    ```bash
    git clone [repository-url]
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Development
Start the local development server:
```bash
npm run dev
```

### Build
Generate a production-ready build:
```bash
npm run build
```
